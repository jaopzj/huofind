# Goofish Miner (Huofind) 🐟

Uma ferramenta moderna para mineração e análise de produtos da plataforma Goofish (Xianyu).

## 🚀 Funcionalidades
- **Mineração em Tempo Real:** Extração de produtos diretamente dos perfis dos vendedores.
- **Análise de Vendedores:** Avaliação automática de confiança baseada em métricas da plataforma.
- **Tradução Inteligente:** Títulos de produtos traduzidos automaticamente para Português.
- **Badges de Identificação:** Detecção automática de versões (EUA, China) e status de bloqueio (Desbloqueado, RSIM).
- **Conversão de Moeda:** Toggle para visualização de preços em Yuan (¥) ou Real (R$).
- **Filtros Avançados:** Filtragem por palavra-chave, preço, modelo de iPhone e armazenamento.

## 🛠️ Tecnologias
- **Frontend:** React, TailwindCSS v4, Vite.
- **Backend:** Node.js, Express, Playwright (Scraper).
- **API de Câmbio:** ExchangeRate-API.

## 📦 Como rodar

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev` (Inicia na porta 3001)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev` (Inicia na porta 5173/5174)
